 SCOB_RAG Directory Structure

  SCOB_RAG/
  ├── src/                       # Source code
  │   ├── pdf_processor.py      # PDF text extraction & chunking
  │   └── weaviate_manager.py   # Weaviate database operations
  ├── config/                    # Configuration
  │   └── rag_config.py         # System settings
  ├── docs/                      # Documentation
  │   ├── README.md             # Full documentation
  │   └── QUICKSTART.md         # Quick start guide
  ├── ingest_documents.py        # Data ingestion script
  ├── rag_query.py              # Query interface
  ├── requirements.txt           # Python dependencies
  └── docker-compose.yml         # Weaviate Docker setup

  Quick Start

  1. Navigate to the folder:
  cd SCOB_RAG
  2. Install dependencies:
  pip install -r requirements.txt
  3. Start Weaviate:
  docker-compose up -d
  4. Ingest documents:
  python ingest_documents.py
  5. Query the system:
  python rag_query.py

  Key Features

  - PDF Processing: Extracts text from SCOB 2015 legal PDFs
  - Smart Chunking: 1000-character chunks with 200-character overlap
  - Local Embeddings: Uses sentence-transformers (no API key needed)
  - Vector Search: Weaviate for fast semantic search
  - Easy Configuration: All settings in config/rag_config.py
